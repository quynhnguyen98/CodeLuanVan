-Khởi tạo
	git init (mở cmd ở trong thư mục, gõ lệnh trực tiếp vào đó)
-Kiểm tra xem có file nào đã sẵn sàng commit hay không
	git status
-Nếu chưa thì 
	git add *
-Kiểm tra lại 1 lần nữa đã add hay chưa
	git status
-Commit lên local respon
	git commit -m "..."
	Nếu bị nhập user để commit
	git config --global user.email "you@example.com"
  	git config --global user.name "Your Name"
	Sau đó nhập lại lệnh commit

-Kiểm tra xem những file commit đã vào local respon chưa
	git log
-Sau đó đồng bộ source code vào origin
	git remote add origin https://github.com/quynhnguyen98/GiaPha.git
-Push lên origin
	git push -u origin master

-------------------------------------------------------------------------------------------------------------------

-Trường hợp nếu code mình không có những lệnh trong source đã commit thì pull về
	git pull
-Xóa lịch sử commit trong github
	git rebase -i ......(id log)
-Recover data (tạo nhánh mới để khôi phục dữ liệu)
	git branch recover .....(id log)
	git checkout recover
	git push --set-upstream origin recover
-Xóa nhánh 
	git branch -d (tên nhánh)
	git push origin --delete (tên nhánh)
